Flow refactor completed.

Main changes:
- Moved the main experiment orchestrator from baseline/mask_iter.py to pipeline/experiment.py.
- Created a new methods/ package:
  - methods/common.py
  - methods/attention.py
  - methods/random_mask.py
  - methods/context_cite.py
  - methods/at2.py
  - methods/recompute.py
- Updated pipeline/runner.py so the default dispatch now imports run_full_pipeline from RagAdaptation.pipeline.experiment.
- Replaced baseline/mask_iter.py with a thin compatibility wrapper that re-exports the old public names.
- Updated baseline/mask_iter_recompute_attention.py to import shared helpers from methods/common.py and core/models.py instead of depending on the old baseline/mask_iter implementation.

What the main flow is now:
run_pipeline.py -> pipeline.runner.run_dataset(...) -> pipeline.experiment.run_full_pipeline(...)

Inside pipeline.experiment.run_full_pipeline(...):
- attention -> methods.attention.run_attention_method(...)
- random -> methods.random_mask.run_random_method(...)
- context_cite -> methods.context_cite.run_context_cite_method(...)
- at2 -> methods.at2.run_at2_method(...)
- recompute -> methods.recompute.run_recompute_method(...)

Compatibility preserved:
- Old imports from RagAdaptation.baseline.mask_iter still work.
- baseline/mask_iter.py now exists only as a wrapper layer.
